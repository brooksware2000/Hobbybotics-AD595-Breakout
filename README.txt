Hobbybotics-AD595-Breakout-V1.0
Hobbybotics AD595 Thermocouple Breakout V1.0

This repository contains a breakout interface for the Analog Devices AD595 Thermocouple integrated circuit.

The board files was created using DesignSpark PCB.

To Use, copy the AD595 library folder into the Arduino\Libraries directory
